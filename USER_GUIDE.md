# TigerEx - User Guide
## Complete Features & Functionality for Users

---

## 🔐 Account Management

### Registration
1. Visit https://tigerex. exchange (or your domain)
2. Click **Sign Up**
3. Enter email and password (min 8 chars, 1 uppercase, 1 number)
4. Verify email via link sent
5. Complete KYC verification (required for trading)

### Login
1. Enter email and password
2. Complete 2FA if enabled
3. Access dashboard

### Password Reset
1. Click **Forgot Password**
2. Enter registered email
3. Reset via email link
4. Set new password

### Profile Settings
- Update email, password
- Configure 2FA
- Set anti-phishing code
- Update display name

---

## 💰 Wallet Management

### Deposit
1. Go to **Wallet**
2. Select cryptocurrency (e.g., BTC, ETH, USDT)
3. Copy deposit address or scan QR code
4. Send funds to address

### Withdrawal
1. Go to **Wallet**
2. Select cryptocurrency
3. Enter destination address
4. Enter amount
5. Complete security verification (2FA, email)
6. Confirm withdrawal

### Transfer
1. Internal transfer to other TigerEx users
2. Zero fees for internal transfers
3. Instant processing

### Transaction History
- View all deposits, withdrawals, transfers
- Export to CSV/Excel
- Filter by date range

## 📈 Trading Features

### Spot Trading
1. Go to **Markets**
2. Select trading pair (e.g., BTC/USDT)
3. Choose order type:
   - **Market**: Execute immediately at market price
   - **Limit**: Set specific price
   - **Stop-Loss**: Auto-sell at loss limit
4. Enter amount
5. Confirm order

### Futures Trading
1. Go to **Futures**
2. Select USDT-M or COIN-M
3. Choose leverage (1x-125x)
4. Open long or short position
5. Manage positions in open trades

### Options
1. Go to **Options**
2. Select underlying asset
3. Choose strike price
4. Set expiration
5. Buy call/put options

### Margin Trading
1. Go to **Margin**
2. Enable isolated/cross margin
3. Borrow funds
4. Open leveraged position
5. Repay with interest

### P2P Trading
1. Go to **P2P**
2. Browse ads (buy/sell)
3. Select payment method
4. Create order
5. Complete payment
6. Release crypto

### Copy Trading
1. Go to **Copy Trading**
2. Browse top traders
3. View performance stats
4. Copy trader (allocate funds)
5. Auto-copy all trades
6. Stop/unfollow anytime

## 💡 Tips

### Security
- Enable 2FA immediately
- Use unique password
- Don't share API keys
- Verify withdrawal addresses

### Trading
- Start with small amounts
- Use limit orders
- Set stop-losses
- Diversify portfolio

### Fees
- Spot trading: 0.1% maker, 0.1% taker
- Futures: 0.02% maker, 0.04% taker
- VIP levels reduce fees
