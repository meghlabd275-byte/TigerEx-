With **smart order routing**, **cross-chain compatibility**, and **300+ advanced features**, TigerEx Hex delivers Binance-level performance with DeFi innovation and complete decentralization options.

## 🚀 Quick Start

### 🐳 Docker Deployment (Recommended)

```bash
# Clone the repository
git clone https://github.com/meghlabd275-byte/TigerEx-.git
cd TigerEx-

# Start all services
docker-compose up -d

# Check service status
docker-compose ps

# Access the platform
# Web: http://localhost:3000
# Admin: http://localhost:3001
# API: http://localhost:8000
```

### 🔧 Manual Installation

```bash
# Backend setup
cd backend
pip install -r requirements.txt
python main.py

# Frontend setup
cd frontend
npm install
npm run build
npm start

# Mobile app setup
cd mobile-app
npm install
npx react-native run-ios    # for iOS
npx react-native run-android # for Android
```

### ☁️ Cloud Deployment

```bash
# Deploy to AWS/GCP/Azure
./deploy.sh production

# Or use Kubernetes
kubectl apply -f devops/kubernetes/deployment.yaml

# Monitor deployment
kubectl get pods -n tigerex
```
