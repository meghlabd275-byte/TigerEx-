## 1. Getting Started

### What is a Wallet?

A digital wallet that allows you to store, send, and receive cryptocurrencies. TigerEx supports two types of wallets:

**Custodial Wallets** - Managed by TigerEx
- Easy to use
- Recovery available
- Built-in support

**Non-Custodial Wallets** - You control
- Full control of private keys
- Import existing wallets
- Advanced features

### Accessing Your Wallet

1. Go to https://tigerex.com
2. Click "Wallet" in the navigation
3. Login or create account
4. Select "Create Wallet"

## 2. Creating a Wallet

### Step-by-Step

**Step 1:** Select Network
```
Supported Networks:
- Ethereum (ETH)
- BNB Smart Chain (BNB)
- Polygon (MATIC)
- Arbitrum (ETH)
- Avalanche (AVAX)
- Solana (SOL)
- TigerEx (TIG)
```

**Step 2:** Choose Type
```
Custodial:
✓ Easier backup
✓ Customer support
✓ Faster transactions

Non-Custodial:
✓ Full control
✓ No third party risk
✓ Hardware wallet compatible
```

**Step 3:** Confirm
```
Your wallet address: 0x742d35Cc6634C0532925a3b8D0D2C8D3D5A5B5B5
Network: Ethereum
Type: Custodial
```

### Backup Your Wallet

**Custodial Wallets:**
- Your wallet is backed up on our servers
- Can be recovered with account credentials
- Contact support if needed

**Non-Custodial Wallets:**
- Save your private key securely
- Write down seed phrase
- Store in safe location
- NEVER share with anyone

## 4. Depositing Funds

### Find Deposit Address

1. Go to Wallet
2. Select network
3. Click "Deposit"
4. Copy address or scan QR

### Deposit Process

**Ethereum/ERC-20:**
```
1. Send tokens to your deposit address
2. Wait for 12 block confirmations
3. Balance updates automatically
```

**BNB Smart Chain:**
```
1. Send BNB to deposit address
2. Wait for 15 confirmations
3. Balance updates
```

**Solana:**
```
1. Send SOL to address
2. Wait for finality
3. Balance updates
```

### Deposit Troubleshooting

| Issue | Solution |
|-------|---------|
| Balance not updating | Wait for confirmations |
| Wrong network sent to | Contact support |
| Deposit not received | Check blockchain explorer |

## 6. Sending Crypto

### Sending Process

1. Select wallet
2. Click "Send"
3. Enter recipient address
4. Enter amount
5. Set gas price (optional)
6. Review and confirm

### Transaction Speed

```
Gas Price    | Speed       | Cost
------------|------------|-------
Slow        | 10+ min    | Low
Normal      | 3-5 min    | Medium
Fast        | < 1 min    | High
```

### Cancelling Pending Transaction

If stuck for >1 hour:
1. Go to transaction details
2. Click "Speed Up" or "Cancel"
3. Sign new transaction with higher gas

## 10. Troubleshooting

### Common Issues

**Transaction Stuck**
- Wait for network confirmation
- Try "Speed Up" option
- Contact support if >24 hours

**Wrong Address**
- Cannot reverse Transactions
- Contact support immediate
- Provide transaction hash

**Balance Incorrect**
- Wait for block confirmations
- Clear cache
- Refresh page

### Contact Support

- Email: support@tigerex.com
- Telegram: @TigerExSupport
- In-app chat
